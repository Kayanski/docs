# Creating additional Credit Accounts

In this tutorial, we will guide you on how to create and manage multiple credit accounts using the Mars dApp. Establishing several accounts allows for more flexibility and better organization of your assets. Let's delve into the steps involved:

### Step 1: Navigation to Credit Account Tab

Begin by clicking on the "Credit Account" tab located at the top of the page. Once there, select "Create Account" to initiate the process of establishing a new credit account.

<img src={ Step1 } style={{ marginBottom: 15 } } />

### Step 2: Transaction Authorization

To proceed, you'll need to authorize the creation of the new account. Do this by signing and confirming the transaction as prompted by the system. 

<img src={ Step2 } style={{ marginBottom: 15 } } />

### Step 3: Funding Your New Account

Now it's time to fund your newly created account. Choose the assets you wish to deposit and specify the amount to transfer into the account. Ensure the details are correct before moving to the next step.

<img src={ Step3 } style={{ marginBottom: 15 } } />

### Step 4: Accessing Your New Account

Upon successful completion of the transaction, you can view your new account from the "Credit Accounts" tab. This section allows you to quickly oversee the details pertaining to your various credit accounts.

<img src={ Step4 } style={{ marginBottom: 15 } } />

### Step 5: Portfolio Overview

If you have set up multiple complex accounts, managing them becomes more streamlined using the "Portfolio" tab. This feature offers an overview of your activity across all accounts, facilitating better organization and monitoring of your assets.

<img src={ Step5 } style={{ marginBottom: 15 } } />
